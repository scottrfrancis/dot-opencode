# Markdown Formatting

When generating Markdown content, always ensure proper spacing for readability:

## Bullet Lists
- Add a blank line before and after bullet lists
- Each bullet item must be on its own line
- Use consistent bullet markers (-, *, or +)

## Paragraphs
- Separate paragraphs with a blank line
- Do not run sentences together without line breaks
- Use double newlines between distinct sections

## Example Format

This is a paragraph.

This is another paragraph separated by a blank line.

Here's a bullet list:

- First item
- Second item
- Third item

And here's text after the list.
